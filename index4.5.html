Прикладное программное обеспечение

Прикладное программное обеспечение можно разделить на два типа: общего назначение и специализированное, предназначенное для выполнения функций определённых отделов.

ППО ОБЩЕГО НАЗНАЧЕНИЯ

Microsoft Office XP Professional

В качестве прикладного программного обеспечения общего назначения используется пакет программ Microsoft Office XP, включающий в себя следующие программные продукты:
-Word - функциональный текстовый редактор
-Excel - приложение для работы с электронными таблицами
-Outlook - почтовый клиент
-PowerPoint - программа для создания презентаций
-Access - работа с базами данных.
Стоимость пакета: 350$

Выбор из трёх программ

Из трех программ (TourManager, Columbis, Мастер-Агент) нужно осуществить выбор на основе расчёте относительных показателей предпочтительности программ. Для этих целей используется программа Assistant Choice.

С помощью программы Assistant Choice вышеописанные программы были оценены по таким параметрам, как:
> функциональные возможности
> интегрируемость с другими проектами
> стоимость
Согласно результатам анализа более приемлимой является альтернатива Cистема Мастер-Агент.
